Latest Release

The latest release is v0.2.0. This release contains some major internal changes including:

  • Remove JavaScript source From JSON
    This is to allow the application to scale for large code bases. The source is now stored in the report directory under 'original-src'.
  • Re-structure _$jscoverage Object
    This is to elegantly accommodate branch coverage properties and any additional future properties. It removes a mix of coverage-array data and object properties.

See the release notes for more information.
