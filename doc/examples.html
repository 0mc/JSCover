JSCover Server

Run example-server.sh (see below) and then go to http://localhost:8080/jscoverage.html?/.

java -jar target/dist/JSCover.jar --document-root=doc/example --report-dir=target

JSCover File Instrumentation

Run example-filesystem.sh (see below) and then go to QUnit test suite.

java -jar target/dist/JSCover.jar -fs --no-instrument=test doc/example-qunit/src doc/example-qunit/out
