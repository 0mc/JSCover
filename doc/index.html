JSCover

JSCover is a tool that measures code coverage for JavaScript programs. It is a Java implementation of the excellent JSCoverage tool, and much of the documentation currently available at that site is applicable to JSCover.

JSCover is free software, distributed under the GNU General Public License version 2.

Why choose JSCover

Reasons that you might choose to use JSCover are:

  • You need coverage measured via a browser (e.g. your tests include DOM interaction).
  • JSCover will run anywhere Java does (NB: JSCoverage may not run on all your platforms).
  • Quick and easy to set-up
  • These limitations/To-Dos don't affect you.
  • JSCover is under active development (see future)

In general though, if you can run JSCoverage, it is the superior tool if you use ECMAScript Language Specification features currently not supported by this tool.

Current State

JSCover is ready to use in server mode, but currently does not support 100% ECMAScript Language Specification. JSCover is currently beta quality, but from little things...

If running as part of a test suite, the test suite should be run a second time without instrumentation to ensure consistency.
