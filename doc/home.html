JSCover

JSCover is a tool that measures code coverage for JavaScript programs. It is a Java implementation of the excellent JSCoverage tool, and much of the documentation currently available at that site is applicable to JSCover. See the file-based version in action now using the QUnit test suite.

JSCover is free software, distributed under the GNU General Public License version 2.

Why choose JSCover

Reasons that you might choose to use JSCover are:

  • You need coverage measured via a browser (e.g. your tests include DOM interaction)
  • You need coverage measured in headless mode (see HtmlUnitServerTest, HtmlUnitFileSystemTest)
  • JSCover will run anywhere Java does (NB: JSCoverage may not run on all your platforms)
  • You need to collate coverage results from multiple independent tests.
  • You need reports stored on the file-system.
  • Quick and easy to set-up
  • JSCover is under active development
  • JSCover is architecturally simple and modular, and so should be easy to contribute to and maintain, hopefully ensuring a long life-time.

If you find a bug please report it and we'll endeavour to correct it. Alternatively, you can try JSCoverage (N.B. JSCoverage does not support branch coverage).

If running as part of a test suite, the test suite should be run a second time without instrumentation to ensure consistent behaviour.

Contribute

Please feel free to contribute - see the To-Do list and github issues. The difficulty level for contributing patches varies depending on the area.

Rhino bugs can be posted there, but please notify us to avoid duplication. If the report is urgent and includes a patch, it can be manually patched and included in JSCover.
